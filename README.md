# Rep Up User Manual

Rep Up is a timer app for ladder-style workouts, which means the numer of reps increases set by set.

## Ladder Workout Principle (Short)

A ladder workout changes reps set by set, usually going up step by step (for example 1, 2, 3, 4...) and then optionally down again.
The goal is to manage effort and rest while accumulating quality reps over a timed session.

## Core Flow

1. Set your total workout time.
2. Press `Start`.
3. Complete the current rep target shown in `Reps`.
4. End the set with `Up` or `Down`.
5. Take the break countdown.
6. Repeat until the workout timer ends.

## Break Modes

### Auto

- Break duration is based on how long your previous set took.
- Example: if your set took about 14 seconds, your next break is about 14 seconds.
- This adapts naturally to your pace and fatigue.

### Fix

- Break duration is calculated from `Base Break (seconds)` x current reps.
- Example: base break `5` and reps `4` => break is `20` seconds.
- This gives a predictable, structured rest pattern.

## Up / Down Controls

During exercise, use these buttons to log the completed set and choose the next direction:

- `Up`: finish current set and increase next rep target.
- `Down`: finish current set and decrease next rep target.
- At rep `1`, `Down` behaves as `Keep` (stays at `1`).

During break, `Up` and `Down` set the direction for the next set.

## Swipe Controls

You can use gestures instead of tapping:

- Swipe up = same action as `Up`.
- Swipe down = same action as `Down`.

This is useful when your hands are sweaty or when you want one-handed control.

## Notes

- Default workout time is set to 7:30 min
- `Pause` simply set the current workout on hold.
- Auto mode is the default mode.

## Inspiration / Reference

- This app is inspired by ladder-style bodyweight training concepts from *You Are Your Own Gym* by Mark Lauren.
